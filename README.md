Колекція готових Docker Compose конфігурацій для різних сервісів та проектів з моїх відео туторіалів.

---

## Як користуватися?

1. Скачай потрібну папку із сервісом із цього репозиторію.
2. Перейди в папку і перевір `docker-compose.yml` та `.env.example`.
3. Перейменуй `.env.example` у `.env` та налаштуй свої змінні (порти, паролі, API ключі тощо).
4. Запусти команду:
```bash
docker compose up -d
